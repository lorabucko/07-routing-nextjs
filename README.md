
# NoteHub app 
+ Sidebar
+ Filter
+ Modal
## ## Tech stack

- React
- Next.js
- TypeScript
- Axios
- TanStack Query
- Formik
- Yup

Проєкт створено за допомогою Next.js (App Router) - це багатосторінкова структура з використанням маршрутизації Next.js.
NoteHub app - застосунок для зберігання, створення, видалення та пошуку нотаток.
- Паралельні маршрути для фільтрації нотаток за тегом
-  Sidebar для фільтрації
-  Модальне вікно з деталями нотатки
 
Для роботи з колекцією нотаток використовується готовий бекенд.
Документація до якого доступна за посиланням:
